Deploy server.js to Railway

Push your server.js, tiktok-gift-alert.html and package.json to a public GitHub repo. Then connect it to Railway (railway.app) for free hosting with no cold starts.

npm install express cors tiktok-live-connector
2

Paste your Railway URL here

Once deployed, copy your Railway app URL and paste it in the Connect tab on the left. The dashboard will auto-generate your OBS URL.

3

Add Browser Source in OBS

In OBS, add a Browser Source. Set the URL to your generated OBS URL below. Set dimensions to 1920×1080, check "Allow transparency", and uncheck "Shutdown source when not visible".

Enter proxy URL above first
4

Go live on TikTok

Start a TikTok LIVE as @hackystreaming. The server auto-detects within 30 seconds and gift alerts will start appearing in OBS. Use this dashboard to preview and customize everything in real-time.

5

Customize everything here

Use the Visual tab to resize cards, adjust effects and animations. Use the Sounds tab to pick different sounds per tier. Use the Rules tab to set coin thresholds and streak milestones. All changes sync to the overlay instantly via the Save & Sync button.
